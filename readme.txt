Powerful chat widget allows you to chat with your visitors on every IM platform including Whatsapp, Skype, Line, Facebook Messenger, SMS and more.
== Description ==
WP Gener8tor is a great WordPress Plugin for your WordPress site that lets your contacts or visitors reach out to you through chat on your website.

Some of the features are listed here:
*This plugin can work with a variety of apps such as Facebook, Whatsapp, Messages, Telegram, Skype, Snapchat and Email. You can set up any one of these on your website or display a widget with all the options.

*The icons will then appear on the website. You can also add a welcome message and a company logo. Further, the plugin allows for a pop up option. You can set a delay time and the icons will pop up after that.

== Installation ==

1. Upload the `WP Gener8Tor` folder to your `/wp-content/plugins/` directory
2. Activate the plugin using the `Plugins` menu in WordPress
3. Go to `WP Gener8Tor` menu from your sidebar.
4. When you log in the first time you will need to provide your name and email id for the activation code.
5. The process is complete
